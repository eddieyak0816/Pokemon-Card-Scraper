1) open command line interface in the same folder as your project file

2) enter coordinates.py -i [image path] -l [type 'r' if no is on the right side 'l' if no is on the left side]

3) for example :
	coordinates.py -i D:\poki6.png -l l

4)Use images with coordinates 734 1024 or use images directly from the sets https://pokemontcg.guru/sets for best results.

5) None of the Optical character recognition (ocr) can guarantee 100% reading the text. If the ocr doesnt read the text
it's not programmer's fault, nor the fault in the program.

6) use test images for testing ,thanks
